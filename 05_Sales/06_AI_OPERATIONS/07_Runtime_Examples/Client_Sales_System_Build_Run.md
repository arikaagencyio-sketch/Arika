# Runtime Example: Client Sales System Build Run

## Objective

Build or reconstruct a client sales operating system from scattered materials.

## Agent Route

1. Planning Agent decomposes the build.
2. Market Intelligence Agent reconstructs ICP and market assumptions.
3. Customer Psychology Agent maps pain, trust, objections, and decision behavior.
4. Offer Architecture Agent creates value and pricing logic.
5. Revenue Operations and Performance Agent defines CRM, metrics, and reporting.
6. Enablement and Playbooks Agent creates SOPs, scripts, and training assets.
7. AI Orchestration Agent creates prompts, memory, and automation boundaries.
8. Risk Trust and Governance Agent checks approval rules.
9. Multi-Agent Orchestration Agent synthesizes the operating blueprint.
10. Reflection and Quality Agent validates evidence and gaps.

## Output

- Sales OS blueprint
- Agent map
- SOP/playbook backlog
- CRM and KPI structure
- AI operations controls
- Governance and approval rules
- Implementation sequence
