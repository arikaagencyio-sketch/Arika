# Runtime Example: Daily Sales Command Run

## Objective

Move revenue today without losing strategic context.

## Agent Route

1. Executive Intelligence Agent sets top commercial priority.
2. Revenue Operations and Performance Agent reviews pipeline, tasks, KPI risks, and forecast deviations.
3. Demand Generation Agent identifies new opportunity creation actions.
4. Lead Qualification and Discovery Agent ranks active leads.
5. Sales Execution and Closing Agent prepares the highest leverage deal action.
6. Follow-Up and Recovery Agent clears stalled open loops.
7. Reflection and Quality Agent checks confidence and missing data.

## Output

- Top 5 revenue actions
- Deal risks
- Follow-ups due
- New pipeline actions
- Human approvals needed
- Memory updates
