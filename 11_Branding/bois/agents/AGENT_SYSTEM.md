# BOIS Agent System

The Master Orchestrator does not perform specialist work. It classifies tasks, assembles retrieval evidence, activates agents, synchronizes outputs, validates continuity, and persists memory.

## Communication Protocol

Every agent receives:

- Dynamic Brand Context
- retrieved evidence by required source
- client object
- task objective
- governance constraints
- expected memory updates

Every agent returns:

- strategic reasoning
- psychological reasoning
- cultural reasoning
- symbolic reasoning
- operational implications
- risks and contradictions
- recommendations
- memory updates
