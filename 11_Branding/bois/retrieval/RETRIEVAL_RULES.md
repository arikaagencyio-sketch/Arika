# Retrieval Rules

BOIS must retrieve before generation.

## Required Sources

Every execution requires:

- sector intelligence
- audience psychology
- cultural context
- client memory
- previous outputs or explicit no-history state
- governance rules

## Retrieval Sources

- sector intelligence
- psychographic intelligence
- cultural intelligence
- client memory
- campaign history
- competitor intelligence
- symbolic systems
- narrative systems
- governance systems

## Failure State

If required evidence is missing, the orchestrator must stop with a retrieval failure and identify the missing source. It must not generate generalized branding output.

## Evidence Shape

Every retrieval result should preserve:

- source document
- paragraph id or memory event id
- tags
- excerpt
- reinforcement count
- retrieval mode
- score

## Generation Rule

Specialist agents may recommend, synthesize, and execute only after the Retrieval Gate confirms all required sources are present.
