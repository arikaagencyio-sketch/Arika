# UI Layer

The BOIS UI should be an operating console for brand cognition, not a logo prompt box.

## Primary Views

- Client Intelligence
- Dynamic Brand Context
- Retrieval Evidence
- Agent Activation
- Brand Memory
- Campaign Systems
- Visual Systems
- Narrative Systems
- Governance Audits
- Brand Scoring
- Evolution Signals

## Interaction Rules

- Show retrieval evidence before generated outputs.
- Show activated agents and their responsibilities.
- Surface governance failures clearly.
- Make memory updates visible and reviewable.
- Keep sector, culture, audience, sales, marketing, and operations connected in the same workspace.
