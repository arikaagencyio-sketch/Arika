# RAG Pipeline

## Pipeline

1. Recursive source ingestion.
2. Paragraph preservation.
3. Taxonomy tagging.
4. Repetition mapping.
5. Knowledge graph construction.
6. Local hybrid retrieval.
7. Vector embedding attachment.
8. Retrieval gate validation.
9. Agent context assembly.
10. Memory update.

## Retrieval Before Generation

Every execution must retrieve:

- sector intelligence
- audience psychology
- cultural context
- client memory
- previous outputs
- governance rules

## Source Preservation

Every retrieved unit preserves:

- document id
- paragraph id
- ordinal position
- original text
- tags
- section hint
- repetition count
- source path

## Strategic Repetition

Repeated paragraphs are preserved and mapped. Repetition is treated as possible strategic reinforcement, not automatic noise.
