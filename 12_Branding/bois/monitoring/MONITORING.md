# Monitoring

BOIS monitors system quality, retrieval health, governance failures, and brand evolution signals.

## Runtime Metrics

- ingestion document count
- ingestion paragraph count
- retrieval completeness rate
- missing retrieval sources
- activated agent coverage
- governance failure count
- brand score trend
- memory update count
- repeated intelligence groups

## Evolution Metrics

- market shift indicators
- cultural shift indicators
- semantic drift
- platform behavior changes
- campaign performance signals
- stakeholder feedback
- audience perception changes

## Alert Conditions

- retrieval failure
- governance critical finding
- score below threshold
- contradiction between brand promise and operational delivery
- cultural risk
- visual or narrative drift
