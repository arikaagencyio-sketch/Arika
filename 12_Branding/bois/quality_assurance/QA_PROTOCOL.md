# Quality Assurance Protocol

BOIS QA checks whether a brand output is usable, specific, governed, and memory-safe.

## Checks

- Retrieval evidence is present.
- Specialist agents were activated.
- Output explains strategic, emotional, symbolic, visual, psychological, cultural, and positioning rationale.
- Output maps to sales and marketing systems.
- Output maps to operations and delivery reality.
- Governance findings are resolved or explicitly accepted.
- Memory updates are ready before execution closes.

## Failure Handling

If a critical issue appears, return the work to the relevant agent instead of smoothing it over during final synthesis.
