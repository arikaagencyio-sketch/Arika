## Prompt templating mechanic — confirmed

The owner clarified directly: the OEOS prompt's Phase 1 "Offer Constraints" block — the short `Category | Details` table plus Monetizable Components and Pricing Potential lists, appearing right before "PHASE 2 — OFFER POSITIONING ARCHITECTURE" — **is the one part of the prompt that changes per offer run**. It functions as the seed brief: a condensed offer summary (likely sourced from this department's own earlier draft material, e.g. an offer-catalog document) that the rest of the 12-phase prompt expands into the full operational system. Everything else in the prompt template (the phase structure, the core operating principle, the final output structure) is identical across all 4 offers captured so far. This confirms the OEOS prompt is a true reusable template with a single per-offer input slot, not a prompt that gets hand-edited each time — useful to know for future offers: the "Offer Constraints" block is the actual brief to look for/request if an offer arrives without its seed data attached.

## Offer #4: Demand Generation Infrastructure (Marketing Division)

**Offer Identity:** Consistent, predictable inbound demand — a self-sustaining, measurable, scalable lead generation engine (funnels, campaigns, acquisition systems, content systems). Problem solved: unpredictable lead flow, founder-led-sales dependency, broken attribution, leaky funnels. Transformation: sporadic campaign-dependent demand → predictable always-on acquisition machine. Target client: $5M-$100M+ B2B/B2C companies with existing product-market fit but scaling bottlenecks. Time-to-result: 30-90 days (initial lead flow), 90-180 days (fully optimized engine). Success criteria: 2x+ inbound lead volume within 90 days, full cross-channel attribution, 20%+ CAC reduction by month 6.

**Pricing (seed brief, Phase 1 only — no tiered breakdown received yet):** Setup $10,000-$75,000. Retainer $5,000-$50,000/month. **This is the widest raw pricing range of any offer captured so far** (offer #1: $7,500-$50,000 setup across 3 tiers; offer #2: $3,500-$20,000; offer #3: $5,000-$100,000+/mo retainer across 4 tiers) — not yet broken into segmented tiers the way offers #1 and #3 were. Flag for the Offer Engineering Registry once/if a tiered breakdown arrives.

**Positioning:** "Premium demand infrastructure provider — we don't run campaigns, we build engines." Core promise: "We will build you a demand engine that produces leads while you sleep — predictable, attributed, and scalable." Core narrative explicitly contrasts "campaigns like a slot machine" vs. "infrastructure" — same systems-over-tactics framing pattern used across offers #1-3 (e.g. offer #1's "Revenue System Architecture" vs. ad-hoc sales activity). Risk reversal: 90-day performance audit included, first 30 days dedicated to setup/attribution so ROI is measurable from day one.

**Backbone — 7 core systems:** Funnel Infrastructure System, Attribution & Analytics System, Paid Acquisition System, Content & Organic System, Lead Nurturing System, Creative Testing System, Reporting & Dashboard System.

**Immutable components (10):** attribution-first architecture, multi-channel approach (no single-channel dependency), structured testing framework, non-negotiable weekly optimization cycle, monthly strategic review, always-on lead nurturing automation, transparent real-time client dashboard, infrastructure-over-tactics (system not one-off campaign), audit-before-execution, quality control checks on every deliverable.

**Customization components:** channel mix, budget allocation, funnel complexity, content strategy, creative approach, nurture sequence depth, attribution model, reporting depth, timeline, technology stack, team size — all explicitly varying by industry/niche/company size/goals/budget/maturity/growth stage, same customization-axis list used across offers #1-3.

**Client journey — all 17 stages specified** (Discovery → Qualification → Audit → Strategy → Proposal → Agreement → Onboarding → Asset Collection → Implementation → Review → Optimization → Delivery → Reporting → Offboarding → Retention → Referral → Expansion), with day-numbered timeline running Day 1 (Discovery) through Day 135+ (continuous Optimization). Notable: this offer's stage list runs Review (Day 120-135, first 30 days post-launch data) as a distinct stage from Optimization — a finer-grained split than offers #1-3 used at the same point in their journeys.

**Internal execution journey — 4-phase, week-numbered (not day-numbered like offers #1-3):** Phase 1 Discovery & Audit (Week 1-4) → Phase 2 Strategy & Planning (Week 4-6) → Phase 3 Implementation (Week 6-12, with foundation setup/funnel-campaign build/creative production/nurture setup/dashboard config/testing/launch sub-steps overlapping in parallel) → Phase 4 Optimization (Week 12-16 initial, then continuous). Includes a Mermaid task-dependency flowchart, a 9-role team-ownership/handoff table (Client Partner → Growth Strategist → Implementation Lead → Analytics Engineer / Campaign Manager / Content-Creative Lead / Email-Nurture Specialist → Performance Manager → back to Client Partner, plus a QA Specialist), a quality-checks table (7 check points: data integrity validation, funnel QA, creative approval, campaign QA, nurture QA, reporting accuracy, performance review), a 5-gate approval-gate table, an **8-row escalation-rules table** segmented by issue type (Technical/Platform, Client Relationship, Performance/KPI Miss, Scope Creep/Budget, Creative/Quality, Compliance/Legal, Resource/Capacity, Client Payment) each with Level 1/2/3 owner and response-time SLA (4hr for technical/compliance, 24hr for relationship/scope/payment, 48hr for performance/resource) — the most granular escalation framework of the 4 offers captured so far, comparable in depth to offer #3's 4-level escalation path but organized by issue-type rather than severity-level. An 8-item automation-opportunities table and a 6-report reporting-requirements table (with format/frequency/audience/owner/delivery-method columns) round out Phase 5.

**Team roles used:** Client Partner, Growth Strategist, Implementation Lead, Analytics Engineer, Campaign Manager, Content/Creative Lead, Email/Nurture Specialist, Performance Manager, QA Specialist — **a fourth distinct (but overlapping) role roster**, different again from offers #1 (Sales Lead, Solutions Architect, Sales Ops Analyst, Data Analyst, Tech Lead, Content Specialist, Project Manager, QA Analyst), #2 (Account Executive, Onboarding Specialist, Operations Analyst, Sales Analyst, Sales Strategist, Project Manager, Sales Enablement Lead, Content Designer, Legal), and #3 (Agency Lead, Strategy Lead, Infrastructure Specialist, Copywriter, List Specialist, Operations Specialist, Inbox Manager, Account Manager).
