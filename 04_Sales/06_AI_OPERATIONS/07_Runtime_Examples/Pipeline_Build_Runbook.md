# Pipeline Build Runbook

Generated: 2026-05-12

Use this when the immediate business objective is to build pipeline for targeted sectors, identify decision makers, start outreach, and create nurturing sequences.

```text
Build a pipeline for now.

Target sectors:
- [sector 1]
- [sector 2]
- [sector 3]

Offer:
[what we sell / what outcome we create]

Geography:
[where]

[company size, buyer type, trigger, pain]

Output I need:
1. Sector prioritization
2. Decision-maker map
3. Lead list criteria
4. Outreach angle per sector
5. First-touch message
6. Follow-up/nurture sequence
7. CRM fields/stages to track
8. Today’s first 5 actions
```

## Output Contract

The completed pipeline build should produce:

- Target sector ranking
- ICP and disqualification rules
- Decision-maker titles
- Buyer pain map
- Trigger events to watch
- Lead source plan
- Outreach angles
- First-touch message
- Follow-up sequence
- Discovery questions
- CRM fields and pipeline stages
- Daily action list
- Approval and risk notes

## Daily Pipeline Action Loop

1. Identify 10 to 25 target accounts.
2. Find decision makers and influencers.
3. Score fit, urgency, and likely pain.
4. Create a sector-specific outreach angle.
5. Send or prepare first-touch messages.
6. Log every account, contact, message, and next step.
7. Follow up based on buyer state.
8. Review replies and objections.
9. Update the sequence and ICP assumptions.
10. Decide tomorrow’s highest leverage action.
